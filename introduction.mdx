# Welcome to RetroWatch API

RetroWatch is a full-stack video processing platform that allows you to upload videos, apply retro shader effects (CRT, VHS, ARCADE), and manage advertisements.

## Features

- **Video Upload & Processing**: Upload videos and apply retro shader effects
- **Advertisement Management**: Upload and manage advertisement content
- **Real-time Processing**: Track video processing jobs in real-time
- **Database Operations**: Full CRUD operations via Supabase integration
- **Secure Authentication**: JWT-based authentication via Clerk

## Base URL

```
https://retrowatch.malhan.ca
```

For local development:

## Getting Started

1. **Authenticate** - Obtain a JWT token from Clerk
2. **Upload a Video** - Use the `/api/protected/videos/upload` endpoint
3. **Process Video** - Apply shader effects via `/api/protected/process-video`
4. **Track Status** - Monitor processing via `/api/protected/processing-status`

## Shader Styles

RetroWatch supports three retro shader styles:

- **CRT**: Classic cathode ray tube monitor effect
- **VHS**: Vintage VHS tape aesthetic
- **ARCADE**: Retro arcade machine styling
